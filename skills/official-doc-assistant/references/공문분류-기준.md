# 공문분류 기준

이 기준은 `data/내업무프로필.md`를 바탕으로 공문을 개인 맞춤으로 분류할 때 사용한다.

## 1. 내 업무 프로필에서 읽을 항목

- 담당 학년, 담당 교과, 담임 여부
- 주 업무, 보조 업무, 담당 행사, 관련 위원회
- 관심 연수, 관심 교육 분야, 수업에 활용하고 싶은 주제
- 내가 직접 처리하지 않는 업무
- 참고만 하면 되는 업무
- 최종 확인자, 함께 확인할 사람

## 2. 분류 범주

| 범주 | 의미 | 복사 위치 |
|---|---|---|
| 직접처리 | 내 업무분장, 담당 학년, 담당 교과, 담당 행사와 명확히 연결되어 내가 처리할 가능성이 높음 | `selected/직접처리/` |
| 공동확인 | 내 업무와 일부 관련되지만 다른 부서, 학년, 관리자 확인이 필요함 | `selected/공동확인/` |
| 참고 | 제출 의무는 낮지만 관심사, 연수, 수업, 정책 이해에 유용함 | `selected/참고/` |
| 전달필요 | 다른 담당자에게 전달하거나 공유해야 할 가능성이 높음 | 기본 복사 안 함 |
| 무관 | 내 업무, 학년, 교과, 관심사와 관련이 낮음 | 기본 복사 안 함 |
| 판단보류 | 제목이나 본문만으로 담당 여부를 확정할 수 없음 | `selected/판단보류/` |

## 3. 직접처리 신호

- 공문 제목이나 본문에 내 주 업무 키워드가 명시됨
- 담당 학년, 교과, 동아리, 행사, 위원회가 직접 언급됨
- 제출물 작성, 신청자 명단 제출, 결과 보고, 설문 응답 등 실행 의무가 있음
- 마감일과 제출처가 명확히 제시됨

## 4. 공동확인 신호

- 내 업무 키워드와 관련은 있지만 담당 부서가 명확하지 않음
- 관리자 결재, 부장교사 확인, 학년 협의가 필요해 보임
- 여러 업무 영역이 걸쳐 있어 한 사람이 단독 처리하기 어려움

## 5. 참고 신호

- 내 관심사와 관련된 연수, 자료, 안내, 정책 소개
- 제출 의무가 없거나 선택 참여 성격
- 수업 아이디어나 학급 운영에 도움이 될 가능성

## 6. 전달필요 신호

- 내 제외 범위에 해당하지만 담당자가 학교 안에 있을 가능성
- 급식, 체육, 방과후, 보건 등 내 프로필상 직접 담당이 아닌 영역
- 특정 담당자나 부서가 명시되어 있음

## 7. 판단보류 신호

- 제목은 관련 있어 보이나 본문에서 담당 여부가 확인되지 않음
- 첨부파일을 읽어야 담당 여부가 판단됨
- 마감일, 제출 방식, 제출물이 불명확함
- 개인정보나 이미지 스캔 때문에 내용을 충분히 읽을 수 없음

## 8. 금지되는 판단

- 근거 없이 담당자를 실명으로 단정하지 않는다.
- 마감일이 없는데 임의 날짜를 만들지 않는다.
- 제출 방식이 불명확한데 공문 회신, 자료집계, 이메일 중 하나로 확정하지 않는다.
- `무관`으로 분류하더라도 근거를 짧게 남긴다.
